# **App Name**: Table Maestro V2

## Core Features:

- Admin Authentication: Login page for administrators using email and password-based authentication.
- Public Booking Page: A public-facing page displays restaurant details and allows guests to make reservations by selecting a date, time, and number of guests.
- Admin Dashboard: Admin dashboard provides a comprehensive overview of table availability, upcoming bookings, and quick actions for managing the restaurant.
- Booking Management: Booking management system to view, search, edit, and cancel reservations, with options to filter and sort by date, time, or guest name.
- Dynamic Restaurant Schedule: Restaurant details form enabling staff to quickly specify the opening hours for specific days of the week.
- AI Waitlist Assistant: AI-powered waitlist management tool that optimizes seating by analyzing reservation data and suggesting seating arrangements.
- Reservation Setting: Settings page providing administrators to set the parameters of time available to reserve, minimum advance to reserve and maximum time duration

## Style Guidelines:

- Primary color: Soft Blue (#77B5FE) to create a sense of trust and tranquility, as well as evoke associations with fine dining.
- Background color: Light gray (#F0F4F8) for a clean, modern look.
- Accent color: Warm Yellow (#FFDA63) for calls to action and key interactive elements.
- Font pairing: 'Playfair' (serif) for headlines and 'PT Sans' (sans-serif) for body text. 'Playfair' is used for headlines due to its stylish and high-end aesthetic, and 'PT Sans' is used for body text because of its modernity.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist and intuitive icons from a consistent icon set (e.g., material design icons). Icons should support wayfinding within the app.
- Employ a grid-based layout for responsive design.
- Subtle transitions and animations on button hovers, form interactions, and page loads to improve UX and provide visual feedback.
